An Open-Source, Procedurally-Generated, Zombie Survival, First-Person Game

This game is still in very early development stages, but once there is a playable version, I will host regular binary builds from my website.

Some of the highlights of the planned game are:

  • Procedurally (and Persistent) Generated Landscapes
  • Procedurally Generated Buildings (who needs models, eh?)
  • Traditional Zombies (only headshots kill!)
  • Perks to keep your character sharp
  • Multiplayer (who can defend themselves against the hord by themselves?)
  • Create barricades - any reasonable object will be fully interactable!
  • And much more...

Being in early development, some of these main features may be modified, but these are my main goals. You can checkout the roadmap for more details as development continues.
